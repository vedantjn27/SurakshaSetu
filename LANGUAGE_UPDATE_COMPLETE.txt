================================================================================
                    KEY LANGUAGE IMPROVEMENTS
================================================================================

ROLE NAMES:
  Admin         → Manager (everyone knows what a manager is)
  Reviewer      → Data Handler (clear job description)
  Analyst       → Viewer (simple and direct)

FEATURE NAMES:
  UBID Registry         → All People
  Data Ingestion        → Upload Data
  Review Center         → Check Matches
  Audit Logs            → Work History
  Privacy Playground    → Tools

ACTIONS:
  Override Status       → Change Status
  Assign to UBID        → Link to Person
  Reclassify Activity   → Update Status
  Split/Merge           → Separate/Combine Records

TECHNICAL TERMS REMOVED:
  "UBID" → "People/Person" (what users care about)
  "Ambiguous Matches" → "Possible Duplicates" (everyone understands)
  "Orphan Events" → "Unmatched Records" (clear meaning)
  "PII Scrambler" → "Hide Information" (explains what it does)
  "Escalate" → "Ask Manager" (tells user what will happen)

COMMON WORDS SIMPLIFIED:
  Submit    → Send
  Edit      → Change
  Delete    → Remove
  Logout    → Sign Out
  Error     → Something went wrong
  Success   → Done!
